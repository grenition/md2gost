# Task 001 — замена буллетов с точки на длинное тире

## Описание

В неупорядоченных списках DOCX сейчас используется маркер в виде точки (●). Нужно заменить его на **длинное тире** (—, Unicode U+2014, em dash).

## Критерии приёмки

- [x] В выводе DOCX для неупорядоченных списков перед каждым пунктом выводится длинное тире (—), а не символ ●.
- [x] Упорядоченные списки не затронуты (нумерация вида `1.`, `2.` остаётся).
- [x] Есть автотест, проверяющий, что маркер неупорядоченного списка — длинное тире.

## Место в коде

- Маркер задаётся в `md2gost/renderable/list.py` в методе `add_item()` (строка с `_ordered else "●"`).
